# Implement-feedback-system

## Overview

Implement-feedback-system is a Java-based Feedback System that allows users to submit feedback for products or services. The system stores feedback records, calculates average ratings, and displays feedback summaries using classes, constructors, and methods.

## Features

- Submit feedback for products or services
- Store feedback records using ArrayList
- Display all submitted feedback
- Calculate average ratings
- Simple menu-driven console application
- Uses Object-Oriented Programming concepts

## Technologies Used

- Java
- OOP Concepts
- ArrayList
- Scanner Class
